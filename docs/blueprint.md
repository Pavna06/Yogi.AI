# **App Name**: Yogi.AI

## Core Features:

- Webcam Pose Detection: Utilize the MoveNet model to detect yoga poses in real-time via webcam. Users can see an overlay on the video of their skeletons to help line up their body.
- Pose Analysis: Analyze the user's pose based on keypoint data from MoveNet and compares to predefined target poses using reasoning to provide appropriate feedback. This tool then displays areas for improvement.
- Audio Feedback: Provide audio feedback using OpenAI's TTS API based on pose analysis.
- Personalized Yoga Plans: Generate personalized yoga plans using the OpenAI API based on user-defined goals.
- Pose Selection: A selector that allows users to choose target poses to guide them through various yoga positions.

## Style Guidelines:

- Primary color: Deep sky blue (#00BFFF) to convey trust and stability in guiding the user.
- Background color: Very light gray (#F0F8FF), a desaturated version of the primary.
- Accent color: Medium slate blue (#7B68EE) for a touch of artistry and creativity, promoting engagement and a sense of personalized instruction.
- Body and headline font: 'Inter' sans-serif for a modern, neutral look
- Code font: 'Source Code Pro' for any snippets related to AI or pose adjustments.
- Use clear and minimalistic icons for guidance and actions.
- Subtle transitions and animations for feedback and pose adjustments to help improve the user experience.
